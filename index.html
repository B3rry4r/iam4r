About Me

MERN Stack Developer

I am a full stack web developer and a Graphics designer, I design branding logos, flyers, banners, wallpapers and more. I make use of some major tools and frameworks used in creating web applications in the modern days. MERN STACK: MERN stands for MongoDB, Express, React, Node, after the four key technologies that make up the stack. MongoDB — document database, Express(.js) — Node.js web framework, React(.js) — a client-side JavaScript framework, Node(.js) — the premier JavaScript web server, using these technologies to create websites provides a very good user experience, thereby attracting more users or customers depending on the content or type of website you want done.
